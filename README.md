## How to run the code
- git clone https://github.com/arun-safalcode/laravel-starter.git
- cd laravel-starter
- open .env and update DB_DATABASE (database details)
- run : `composer update`
- run : `php artisan key:generate`
- run : `php artisan migrate:fresh --seed`
- run : `php artisan serve`

- Best of luck
